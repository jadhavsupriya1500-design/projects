This project is a Tribute Page created using HTML and CSS as part of Level 2 – Task 2.
The page is dedicated to Albert Einstein, showcasing his life, achievements, and contributions to modern physics.
The objective of this project is to:
1.Practice basic HTML structure
2.Apply CSS styling
3.Create a clean layout
4.Learn GitHub Pages deployment
#Technologies Used
-HTML5 – Page structure
-CSS3 – Styling and layout
-GitHub Pages – Hosting the website

# Steps to Create This Tribute Page
Step 1: Create Project Folder
Create a folder and add the following files:
index.html
style.css
images.jpg

Step 2: Write HTML Code
Add basic HTML structure
Create sections for:
Title
Subtitle
Biography text
Image
Link the CSS file using <link>

Step 3: Style Using CSS
Use Flexbox to align text on the left and image on the right
Increase font size for better readability
Add spacing and responsive design
Ensure the page works on mobile devices

Step 4: Upload Files to GitHub
Create a GitHub repository
Upload all project files to the main branch
Ensure file names are correct (index.html, not .txt)

Step 5: Deploy Using GitHub Pages
Go to Settings → Pages
Select:
Source: Deploy from a branch
Branch: main
Folder: / (root)
Save and wait for deployment
